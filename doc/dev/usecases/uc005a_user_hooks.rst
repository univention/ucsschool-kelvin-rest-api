:ref:`UC-005a: User Hooks<uc005a_user_hooks>`
---------------------------------------------

:Actor: Administrators, Operators
:Priority: Must-have

Administrators can add hooks which have side-effects on read, creation, update or delete of a user object.
For example, a hook could be used to add a user to a certain group based on their role.
