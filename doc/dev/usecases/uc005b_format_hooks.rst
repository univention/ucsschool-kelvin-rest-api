:ref:`UC-005b: Format Hooks<uc005b_format_hooks>`
-------------------------------------------------

:Actor: Administrators
:Priority: Must-have

Administrators can add hooks which change format of attributes when they are used for the generation of other attributes.
For example, when generating an email address, the first name might be truncated to a maximum length by the format hook.
